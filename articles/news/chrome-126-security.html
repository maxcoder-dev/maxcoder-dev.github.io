Chrome 126 получил крупное обновление безопасности

Новости · 14.01.2026

Google выпустила стабильную версию Chrome 126, сосредоточенную на устранении уязвимостей и повышении общей безопасности браузера.

Критические исправления

В Chrome 126 закрыто более 20 уязвимостей, включая несколько критических проблем с управлением памятью.

  • исправлены use-after-free уязвимости;
  • усилена изоляция процессов;
  • обновлены sandbox-механизмы;
  • улучшена защита JavaScript-движка V8.

Что это значит для пользователей

Пользователям рекомендуется обновиться как можно скорее, так как часть уязвимостей могла использоваться в реальных атаках.

Для разработчиков

Обновление также включает незначительные улучшения Web API и стабильности DevTools, без ломающих изменений.
